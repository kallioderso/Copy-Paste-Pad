# copy_paste_board

![copy_paste_board](https://raw.githubusercontent.com/kallioderso/Copy-Paste-Pad/refs/heads/main/Images/image7.png)

a secondary Keyboard, made to grande different Copy Paste slots at the same time, for the purpose of more efficient working.

* Keyboard Maintainer: [Pascal Valentin Maximilian Lösch](https://github.com/kallioderso)
* Hardware Availability: Shop is currently in development

Make example for this keyboard (after setting up your build environment):

    make copy_paste_board:default

Flashing example for this keyboard:

    make copy_paste_board:default:flash

See the [build environment setup](https://docs.qmk.fm/#/getting_started_build_tools) and the [make instructions](https://docs.qmk.fm/#/getting_started_make_guide) for more information. Brand new to QMK? Start with our [Complete Newbs Guide](https://docs.qmk.fm/#/newbs).

## Bootloader

Enter the bootloader in 3 ways:

* **Bootmagic reset**: Hold down the key at (0,0) in the matrix (usually the top left key or Escape) and plug in the keyboard
